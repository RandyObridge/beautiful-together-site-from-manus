# Beautiful Together

**Leadership Development by Julian S. Newman**

A clean, standalone static website for the *Beautiful Together* book and leadership development platform. Built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies, no build step required.

## Features

- Fully responsive, mobile-first design
- Dark theme with pink, yellow, and cyan accent palette
- Smooth scroll navigation
- Fade-in animations on scroll
- Sections: Hero, Core Philosophy, Proven Methodologies, Flagship Program, Course Catalog, Bridge-Building Tools, CTA, Footer

## Deployment

This site is ready to deploy directly to **Netlify** by connecting this GitHub repository. No build command or publish directory configuration is needed — Netlify will serve `index.html` from the root.

## Local Preview

Open `index.html` directly in any browser, or run a simple local server:

```bash
npx serve .
```
